為什麼要建立 Hexo 部落格

可以藉由寫文章幫助大家並整合自己的技術。平台這麼多，為何我會選擇 Hexo ？

因為 Hexo 是要用指令來開發，也必須配合 Git 做使用，，是不是摸蜊仔兼洗褲呢！
